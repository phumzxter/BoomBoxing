As a casual fighting game, within boxing theme, by traditional, we come up with two main modes: person versus computer, person versus person, online mode.

How to play the game

This is a straight-forward fighting game but instead of using controller, keyboard, joystick, â€¦ player need to using microphone, player need to shout out command in clearly, normal volume voice, also the command have to speak correctly or it will not perform. For this game, we only support English as main controller language. To control the boxer, we have pre-defined command for each action as bellow:

- Forward: up

- Backward: back

- Left: left

- Right: right

- Left Jab: left jab

- Right Jab: right jab

- Heavy Right Punch: heavy

- Defense: block
