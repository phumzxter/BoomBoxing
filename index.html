Project Specification

Abstract

This project might has not reached its full potential since our group still lack of many experiences in this field, but we hope that this game can be improve in the future since voice control game has an extremely high improving potential in the future. Lucky for us, we have a chance to post our game report on Internet platform. As a result, we will summarize all the material and techniques that we have used in this project for other people in the future to explore more and more about our project, as well as developing the voice control game in the future.

Detail

In this part, we will summarize our project from Unity perspective. We believe that this will help to other to understand our project clearer if they want to have any improvement in the future.

Firstly, the whole assets and material that we used for the project are organized and named accordingly in the assets of the project for the ease of finding information.

The most important folder in term of our project is Scene folder. In this folder, all the scene from online, offline, multiple play as well as the menu are named accordingly.

All animations and animation controllers are saved in animation folder, all the animation have been used in the game are in the same place as well. We have downloaded many different animations sets during the developing process; these files can be found in Frank_fighting folder. Font, material, audio, sprite, are the resources for the games such as color (pic1), music, sound effects, and HUD components.

About the scripts in this project, all are saved in Scripts and all of them are used as a component in different object among different scenes. All of them are written in form of C# code, which will result in faster speed for the project as well as it is the main language in Unity. After experiencing with Unity for around 4 months.
