Game Mechanic

Mechanic of the game

Boom Boxing is the straight-forward fighting games with the only different element is way of controlling. So for this kind of game, each player are represent by an avatar in shape of a graphical boxer. The game is set in theme of boxing arena, within limited time and space. The goal of player is knock out the opponent or defeat with series of actions with well-timing input. Players have the same set of actions, so to winning it depend on your skill

Action

As stated in gameplay section, players are provide with the same set of actions: Punch, Kicking and Defense. Players using these actions to re-action with situation in game. In Boom Boxing we try to make it to be a fun game, we using rock, paper, scissor game as the references to the actions we pre-defined, this action is counter or lose to other actions. Winning conditions

There are two way two win this game, timer and knock out. Each character has a health bar counter (HP), each actions will have a based damage. Each successful action will decline HP of the opponent.

Move Base Damage
Jab 20
Heavy Punch 50

This game having the boxing theme, we try to lend some of the real elements. For each round of the battle, player have 90 second to win. If the player has more HP at the end of the round, player will win, other else, the opponent win. Or in second situation, K.O or knock out, player win by decline all the HP of the opponent without caring by the timer.
